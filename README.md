# MakefileMaker
forfeignasse


HOW TO USE:

`./MakefileMaker [ files...]`	(will take all .c files in the directory you indicates)  
`./MakefileMaker`   			(will take all .c files in the current directory)  

And the Makefile will be created in the current directory you're at.
